    标准数据包

    KWO采用websocket进行通讯，本页面会介绍通讯使用的标准数据包

    📤 KWO发出的数据包

    玩家加入服务器

    ✌️ player join

    {
    "type":"pack",
    "cause":"join",
    "params":{
        "sender":"lition",
    }
}

    玩家离开服务器

    👋 player left

    {
    "type":"pack",
    "cause":"left",
    "params":{
        "sender":"lition",
    }
}

    玩家聊天

    💬 player chat

    {
    "type":"pack",
    "cause":"chat",
    "params":{
        "sender":"lition",
        "text":"哼哼 啊啊啊啊啊"
    }
}

    执行命令返回

    {
    "type":"pack",
    "cause":"runcmdfeedback",
    "params":{
        "result":"执行命令返回",
        "id":"执行时传入的id"
    }
}

    服务器开启

    {
    "type":"pack",
    "cause":"start",
    "params":{}
}

    服务器进程退出

    ⚠️ 此处退出代表进程退出，如果用Bat启动程序则可能程序依旧在后台运行

    {
    "type":"pack",
    "cause":"stop",
    "params":{}
}

    直接发送文本

    {
    "type":"pack",
    "cause":"plantext",
    "params":{
        "text":"哼哼 啊啊啊啊啊啊",
    }
}

    调试信息

    {
    "type":"pack",
    "cause":"debug",
    "params":{
        "msg":"此处返回信息不固定，作为描述使用",
    }
}

    数据包解析失败

    {
    "type":"pack",
    "cause":"decodefailed",
    "params":{
        "msg":"描述文本",
    }
}

    📥 KWO可接收的数据包

    向服务器发送文本消息

    {
    "type":"pack",
    "action":"sendtext",
    "params":{
        "text":"要向服务器中发送的文本",
        "id":"此项无用"
    }
}

    执行命令

    {
    "type":"pack",
    "action":"runcmdrequest",
    "params":{
        "cmd":"要向服务器中执行的命令",
        "id":"返回执行结果时需要附带执行id"
    }
}

    开启服务器

    {
    "type":"pack",
    "action":"start",
    "params":{}
}

    关闭服务器

    {
    "type":"pack",
    "action":"stop",
    "params":{}
}
