    搭建开发环境

    Python

    pip install websockets

    websocket库，用于与KWO或LLWS进行websocket通信

    pip install pycryptodome

    加密库，用于AES加、解密

    JavaScript

    将远程仓库clone到本地后，执行

    npm i

    会自动装载所需的node库

    Lua

    所有Lua使用的api均由NeoLua库注册到Lua虚拟机中

    理论上您只需要.NET Formwork4.7.2环境即可运行
